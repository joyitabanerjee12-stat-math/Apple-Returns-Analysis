# Apple Daily Returns Analysis

This repository contains an analysis of Apple Inc. (AAPL) daily returns, comparing Up vs Down days using ANOVA and T-Test.

## Files
- `Apple_Returns_Analysis.md` → Full report with tables, formulas, and interpretation
- `Apple_Returns.xlsx` → Raw data and Excel calculations

## Overview
- Computed daily returns from closing prices
- Classified days into Up (positive returns) and Down (negative returns)
- Performed One-Way ANOVA and two-sample t-test

## How to Use
1. View the Markdown report for results and interpretations.
2. Open the Excel file to see calculations and formulas.
